# FFWeb

A web gui for FFW (Fuzzing For Worms) fuzzing framework.


## Install

```
sudo pip install django djangorestframework whitenoise
```

## Run in production

```
export DJANGO_SECRET_KEY='<key>'
export DJANGO_DEBUG="" # enables prod
```
